# Urban Stay Finder

🏠 A full-stack rental property web application for discovering long-term and short-term stays worldwide.

## Features

✨ **Auth & Accounts**
- Email/password signup and login
- Google OAuth integration
- Two account types: Renter and Landlord/Agent
- Verified accounts badge for landlords

🗺️ **Property Search & Maps**
- Google Maps integration with property pins
- Advanced filters: City, price range, bedrooms (1-8), property type
- Property types: Single room, studio, apartment, house, hotel, Airbnb

🏠 **Property Details**
- Photo gallery with 360° interior views
- Complete amenities list
- Google Maps embed with nearby locations
- Walking distance calculations
- Landlord profile with verified badge
- Multiple pricing options

📅 **Booking & Payments**
- Calendar-based booking system
- Stripe and M-Pesa payment integration
- Booking history tracking

## Tech Stack

- React 18 + TypeScript + Tailwind CSS
- Supabase Backend
- Google Maps API
- Stripe + M-Pesa Payments
